🐙 How to host on GitHub (Free)
1
Go to github.com → Sign up free if wala ka pa
2
Click + → New repository → name it anything → set Public → check Add README → Create repository
3
Click Add file → Create new file → filename: version.json
4
I-paste yung JSON sa taas (click Copy JSON) → click Commit changes twice
5
Click version.json → click Raw button → copy the URL from browser bar
6
Paste that Raw URL in the field above ↑ — done! 🎉
Looks like: raw.githubusercontent.com/yourname/repo/main/version.json
💡 Para mag-update ng version: Bumalik sa GitHub → click version.json → click ✏️ pencil → baguhin ang version number → Commit. Automatic na mag-aappear ang update prompt sa users!

How to Build Your APK
Two methods — GitHub Actions is easiest
1
Extract the ZIP
Unzip the downloaded file. You'll see folders like app/, gradle/, and .github/.
2
Create a GitHub Repository
Go to github.com/new → create a public repo (public = free Actions minutes)
3
Push files to the ROOT of your repo
⚠️ Upload the contents of the extracted folder — not the folder itself.
✅ repo root should contain: gradlew · build.gradle · app/ · .github/ · gradle/
4
Wait 3–5 minutes
Go to the Actions tab in your repo. The "Build APK" workflow runs automatically and installs all required SDK tools.
5
Download your APK
Click the completed workflow run → scroll to Artifacts → download your APK. Share via Google Drive!
1
Install Android Studio
Download from developer.android.com/studio (free, ~1 GB)
2
Open the Project
File → Open → select the extracted folder
3
Build APK
Menu: Build → Build Bundle(s) / APK(s) → Build APK(s)
4
Find your APK
Located at: app/build/outputs/apk/debug/app-debug.apk
