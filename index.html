How to Build Your APK
Two methods — GitHub Actions is easiest
1
Extract the ZIP
Unzip the downloaded file.
2
Create a GitHub Repository
Go to github.com/new → create a public repo.
3
Push files to the ROOT of your repo
⚠️ Upload the contents of the extracted folder.
4
Wait 3–5 minutes
Go to the Actions tab. "Build APK" runs automatically.
5
Download your APK
Click the completed workflow run → scroll to Artifacts → download your APK.
1
Install Android Studio
Download from developer.android.com/studio
2
Open the Project
File → Open → select the extracted folder
3
Build APK
Menu: Build → Build Bundle(s) / APK(s) → Build APK(s)
